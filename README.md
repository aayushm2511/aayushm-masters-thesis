# aayushm-masters-thesis
Code for my master's thesis -  dealing with response diversity and ecosystem stability in the Western Ghats

This repository will have all the code I write during the course of my master's thesis, as listed below:

1.) rd_calculations.rmd - This is a R Markdown file containing code to estimate species occurrence probability curves along the Western Ghats using Generalised Additive Models. This represents the stacked species distribution model (SSDM) approach to modelling. The file also contains the code to estimate the divergence aspect of response diversity (Ross et al., 2023). The data used is species presence-absence data from Krishnadas et al., 2021 - encompassing 183 species from 286 sites across the Western Ghats.
